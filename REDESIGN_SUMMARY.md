## Design Features

### Color Scheme
- **Primary**: #ec8951 (Orange - brand color)
- **Text**: #222222 (Dark gray)
- **Secondary Text**: #666666, #999999
- **Background**: #ffffff (White)
- **Accents**: #007bff (Blue for CTAs)

### Typography
- **Font Family**: Montserrat (existing)
- **Headings**: Bold (600-700 weight)
- **Body**: Regular (400 weight)
- **Responsive**: Fluid typography using calc()

### Key Design Elements
1. **Modern Card Designs**: Rounded corners, shadows, hover effects
2. **Gradient Overlays**: Used on hero and category images
3. **Smooth Animations**: Transitions on hover and interactions
4. **Clean Layout**: Generous white space, clear hierarchy
5. **Responsive Design**: Mobile-first approach
